# AltaVox Radio

## 🇬🇧 English

**AltaVox Radio** is a modern internet radio streaming application for Android smartphones and Android TV, inspired by Transistor. It delivers stable streaming, dynamic playlist management, and a customizable OneUI-inspired interface optimized for both mobile and television experiences.

> This application is only an `.m3u` playlist player.  
> Users are responsible for preparing and hosting their own playlist files.

---

## 🚀 New Features

- **Global Online Radio Search (radio-browser.info)**  
  Search stations by Name, Country, and City with smart search guidance.

- **Instant Broadcast Preview**  
  Preview online radio streams directly from search results before adding stations.

- **Dynamic Widget Color Hex Picker**  
  Customize widget backgrounds using custom Hex color codes.

- **Direct Widget Playback**  
  Widgets can now start playback directly even when the app is fully closed.

- **Android TV - Glassmorphic Top-Right Player**  
  Elegant mini-player beside the TV clock for a cleaner and more space-efficient layout.

---

## ✨ Main Features

- Dynamic `.m3u` playlist loader
- Android TV optimized interface
- Light, Dark, and System themes
- Custom theme colors for Light & Dark mode
- Samsung OneUI-inspired widget
- Smart radio categorization
- Import & export `.m3u` playlists
- Optimized streaming buffer
- Modern Material Design interface
