## 🇬🇧 English

**AltaVox Radio** is a modern internet radio streaming application inspired by Transistor, designed to provide a lightweight, flexible, and highly customizable listening experience across Android smartphones and Android TV devices. The application focuses on stable streaming playback, dynamic playlist management, Samsung OneUI-inspired aesthetics, and comfortable usability for both mobile and television environments.

> This application is only an `.m3u` playlist player.  
> Screenshots shown are for illustration purposes only.  
> Users are required to independently prepare and host their own `.m3u` playlist files.  
> The developer is not responsible for any playlist content, radio broadcasts, or materials contained within user-provided `.m3u` playlists.

---

## ✨ Main Features

### 🔗 Dynamic M3U Playlist Loader

Load radio playlists dynamically from `.m3u` files hosted on user-owned servers. Playlist changes can be updated anytime without requiring an application update.

### 📺 Dedicated Android TV Experience

Specially optimized interface for Android TV with remote-friendly navigation, customizable theme colors, and a dedicated settings page designed for television screens.

### 🎨 Flexible Theme System

Supports:

- Light Theme
- Dark Theme
- Follow System Theme

### 🌈 Custom Theme Colors

Customize separate color schemes for light and dark modes to match personal preferences.

### 🧩 Samsung OneUI Inspired Widget

Beautiful home screen widget inspired by Samsung OneUI for quick access to 2 favorite radio stations and currently playing radio information.

### 📂 Smart Radio Categorization

Radio stations can be grouped into categories. Favorite categories are automatically prioritized at the top for easier access.

### 📥 Import & Export Playlist

Supports playlist import and export in standard `.m3u` format for easy backup and playlist sharing.

### 📡 Optimized Streaming Buffer

Uses an improved streaming buffer mechanism to minimize buffering issues and maintain stable audio playback.

### 🖼 Improved UI Readability

Enhanced design language for better readability and improved visual consistency aligned with Samsung OneUI and Google Material Design principles.

## ✨ Fitur Utama

### 🔗 Dynamic M3U Playlist Loader

Memuat playlist radio secara dinamis dari file `.m3u` yang dihosting di server milik pengguna. Playlist dapat diperbarui kapan saja tanpa perlu update aplikasi.

### 📺 Pengalaman Khusus Android TV

Antarmuka khusus Android TV yang dioptimalkan untuk navigasi remote, mendukung kustomisasi warna tema, serta dilengkapi halaman Settings khusus untuk tampilan televisi.

### 🎨 Sistem Tema Fleksibel

Mendukung:

- Tema Terang
- Tema Gelap
- Mengikuti Tema Sistem

### 🌈 Kustomisasi Warna Tema

Pengguna dapat mengatur warna berbeda untuk mode terang dan gelap sesuai preferensi masing-masing.

### 🧩 Widget Ala Samsung OneUI

Widget cantik ala Samsung OneUI untuk akses cepat 2 radio favorit dan informasi radio yang sedang diputar langsung dari home screen.

### 📂 Kategorisasi Radio Pintar

Stasiun radio dapat dikelompokkan ke dalam kategori. Kategori favorit akan otomatis diprioritaskan pada bagian paling atas.

### 📥 Import & Export Playlist

Mendukung impor dan ekspor playlist format `.m3u` untuk memudahkan backup dan berbagi playlist.

### 📡 Streaming Buffer Optimal

Menggunakan mekanisme buffer streaming yang lebih optimal untuk mengurangi buffering dan menjaga kestabilan pemutaran audio.

### 🖼 Tampilan Lebih Mudah Dibaca

Peningkatan design language agar lebih nyaman dibaca dan lebih selaras dengan Samsung OneUI serta Google Material Design.
